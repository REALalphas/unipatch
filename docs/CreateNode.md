# CreateNode

## Description
This class defines the `Create` AST node instruction, generated by the `create()` global function. It dictates to the package engine the creation of a brand new configuration or data file in the pipeline's relative output path structure.

## Methods
Unlike `EditNode` or `GetNode`, `CreateNode` currently lacks chainable modification methods and relies entirely on the parameters supplied during its initialization via the `create()` function.

## Properties
* `path` (`string`) - The relative destination path of the file.
* `content` (`any`) - The payload of data that will be populated in the final generated structure.
* `format` (`FileFormat | 'text'`) - The target formatting standard the output stream will serialize to (e.g., `ini`, `json`, `yaml`, `text`).

## Examples

### Generating an Object Literal Configuration
An example representing how `CreateNode` is constructed sequentially through the DSL without subsequent modifications.

```typescript
import { pkg, create } from 'unipatch-engine';

const engine = pkg().put(
    create('system/boot.ini', {
        Splash: {
            enabled: true,
            timeout: 5000
        }
    }, { type: 'ini' })
);
```
